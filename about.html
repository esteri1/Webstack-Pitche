About Haptic Technologies

Haptic Technologies already present on some Tablets and smart Phones is emerging as an innovative way to augument visual information.

The implications for accessibility are exiciting and this project explored using vibrotactile feedback to augument information in HTML content.

Additionally, instructions and source code for adding Haptic Effect to HTML content is provided.

Finally usability issues with haptic interfaces are discussed along with the results from a usability study with students with visual impairments.

Haptic devices may incorporate tactile sensors that measure forces exerted by the user on the interface.
